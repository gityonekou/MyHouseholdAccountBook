# MyHouseholdAccountBook
マイ家計簿のWebアプリケーションです。<br>
SpringBootを使って作成しています。<br>
バージョン1.0.0以降、生成AI(Claude Code)を使っています。<br>
<br>
「develop-1.02」ブランチは、以下機能追加用のブランチです。<br>
【追加機能一覧】<br>
・月別収支確認画面の修正（支出単位の一覧を追加し、支出カテゴリ単位の一覧とタブ切替えで切替とする）：feature-1.02-dev1<br>
・0円固定費を登録可能とし、対象月の初回収支登録時にメッセージ表示、内容確認ボタン押下時に0円から変更されていない支出はエラーメッセージを表示する：feature-1.02-dev2<br>
<br>
<br>
　テストはJUnitで単体・結合まで実施<br>
　DDDを意識しながら、リファクタリングは随時実施します。<br>
　<br>
　＜追加予定の機能＞<br>
　[ver1.03で作成]<br>
　・現金・口座振替・クレジットカード支払い・デビットカード支払いなど、支払方法を登録できるようにする。<br>
　　また、翌月引落のクレジットカード利用代金のうち、指定店舗の利用代金をポイント利用分だけ減算できるようにする。<br>
　[次期未定]<br>
　・買い物登録詳細機能の作成など、ベータ版でメニューだけ作成して、機能の取込が出来ていないものの機能取込<br>
　・イベントに応じた買い物を登録できるようにする<br>
　・買い物登録の残は携帯(アンドロイド側）のアプリもkotlinで作る予定なので、そのSV処理などなど、、、<br>
<br>
