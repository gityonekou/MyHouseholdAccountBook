# MyHouseholdAccountBook
マイ家計簿のWebアプリケーションです。<br>
SpringBootを使って作成しています。<br>
バージョン1.0.0以降、生成AI(Claude Code)を使っています。<br>
<br>
「feature-1.01」ブランチは「固定費登録の改善」用のブランチです。<br>
【追加機能一覧】<br>
・固定費登録ユースケースのリファクタリング実施：feature-1.01-dev1<br>
・同一分類の固定費（同じ支出項目の固定費）の一括更新機能追加：feature-1.01-dev2<br>
・固定費一覧の合計行の表示内容更新：feature-1.01-dev3<br>
・年間固定費合計画面追加：feature-1.01-dev3<br>
<br>
　テストはJUnitで単体・結合まで実施<br>
　DDDを意識しながら、リファクタリングは随時実施します。<br>
　<br>
　＜追加予定の機能＞<br>
　・買い物登録詳細機能の作成など、ベータ版でメニューだけ作成して、機能の取込が出来ていないものの機能取込<br>
　・イベントに応じた買い物を登録できるようにする<br>
　・買い物登録の残は携帯(アンドロイド側）のアプリもkotlinで作る予定なので、そのSV処理などなど、、、<br>
<br>
<br>
【技術スタック】
・SpringBoot:3.0.7<br>
・Spring Security:6.0.7<br>
・Mybatis:2.2.2<br>
